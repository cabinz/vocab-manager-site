# Version 1.0.0
[Download](https://github.com/cabinz/vocab-manager/releases/download/v1.0.0/WindowsInstaller_v1.0.0.zip)

这是 Vocab Manager 的第一个正式发布版本！
支持的功能有：

* 对用户自己生词及其细节的增删改查；
* 对生词进行标签化管理；
* 在回收站恢复自己移除的生词；
* 对自己的词汇库进行遗忘测试；
* 生成测试错误报告 / 单词笔记。
